文字属性（Control Code）保存格式与过程 (2)

各种类型变量的保存方式

没有Byte类型的数据。Word, DWord, QWord 的 spra 分别是0、1、2。枚举类型和布尔值统一都用 Word 类型。字符串采用变长序列表示，即 spra 为3，后跟一个Word表示长度，不要求有0结尾符。由于字符串统一采用Unicode，不用考虑字对齐的问题。
